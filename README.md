# UchuAddon

**UchuAddon** は、  
**Nebula on the Ship** 向けに作られたアドオンです。  
新しい役職やゲーム要素を追加し、プレイ体験を拡張します。

---

## 概要

- 種別：Nebula on the Ship 用Addon
- ベースMOD：Nebula on the Ship
- 内容：追加役職・追加システム

---
